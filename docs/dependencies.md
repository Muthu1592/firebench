# 12. Dependencies

## Required

- h5py < 4.0
- numpy < 3.0
- pint < 1.0
- SALib < 2.0
- scipy < 2.0
- geopandas < 2.0
- matplotlib > 3.8

## Optional
These dependencies are required for advanced use (testing, offline generation of documentation, etc.)

- bandit
- black
- pylint
- pytest
- pytest-cov
- pytest-mock
